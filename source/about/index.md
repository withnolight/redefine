## 关于我

Haskina 是我众多网名中的一个。这个词来自 `Haska`，是我高中时无所事事时想出来的一个奇怪名字。后来把 `-a` 换成 `-ina`，让我感觉更好了一点。

{% grid cols=2 gap=16 class="about-fact-grid" %}
{% callout success fa-solid fa-location-dot :: 此时此地 %}
2005 年的深秋出生于四川。此后辗转于德阳、绵阳、成都之间求学。*到底要不要离开……？*
{% endcallout %}

{% callout info fa-solid fa-flask-vial :: 正在学习 %}
化学工程与工艺专业本科学生，目前也是快要毕业了。*四年江安还是多少有点坐牢了。*
{% endcallout %}

{% callout primary fa-brands fa-linux :: 桌面环境 %}
主力操作系统为 Arch Linux，桌面环境为 KDE Plasma。*要不是某些工程软件需要 Windows™ 的话，早就全面换成 Linux 了。*
{% endcallout %}

{% callout orange fa-regular fa-keyboard :: 输入习惯 %}
双拼使用者，使用的方案为自然码。*新手还是直接去学小鹤吧。*
{% endcallout %}
{% endgrid %}
